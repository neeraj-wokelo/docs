## Authentication

All requests must include an **API key** in the `x-api-key` HTTP header.

```text
x-api-key: <YOUR_API_KEY>
```

<Warning>
  Never expose your API key in client-side code, browser requests, or public repositories. A missing or invalid key returns `401 Unauthorized`.
</Warning>

## Execute your first request

<CodeGroup>

```bash cURL
curl --location 'https://api.akta.pro/api/v1/company/employee-reviews?company=canva.com&limit=10&offset=0' \
  --header 'x-api-key: <YOUR_API_KEY>'
```

```python Python
import requests

response = requests.get(
    "https://api.akta.pro/api/v1/company/employee-reviews",
    headers={"x-api-key": "<YOUR_API_KEY>"},
    params={
        "company": "canva.com",
        "limit":   10,
        "offset":  0
    }
)
data = response.json()
emp  = data["data"]["employee_reviews"]

overall = emp["overall_rating"].get("glassdoor")
total   = emp["reviews"]["total"]
print(f"Overall rating: {overall}/5.0  |  {total} reviews total")
```
