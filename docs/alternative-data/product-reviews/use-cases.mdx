### Fetch product reviews for one or more products

Lets say we want to pull product reviews for Canva. First step would be to fetch the product\_ids. 

<CodeGroup>

```bash cURL
curl --location 'https://api.akta.pro/api/v1/company/product-reviews?company=https%3A%2F%2Fcanva.com' \
--header 'x-api-key: your api key'
```

```python Python
import requests

url = "https://api.akta.pro/api/v1/company/product-reviews?company=https://canva.com"

payload = {}
headers = {
  'x-api-key': 'your api key'
}

response = requests.request("GET", url, headers=headers, data=payload)

print(response.text)
```

**Fetch product reviews for one or more products.**

Include product\_ids in the products parameter while making the API call.

<CodeGroup>

```text cURL
curl --location 'https://api.akta.pro/api/v1/company/product-reviews?company=https%3A%2F%2Fcanva.com&products=89224%2C%20148891' \
--header 'x-api-key: your api key'
```

```text Python
import requests

url = "https://api.akta.pro/api/v1/company/product-reviews?company=https://canva.com&products=89224, 148891"

payload = {}
headers = {
  'x-api-key': 'your api key'
}

response = requests.request("GET", url, headers=headers, data=payload)

print(response.text)
```
