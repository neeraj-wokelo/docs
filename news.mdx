## Overview

The News API fetches news articles filtered by company, industry, date range, sentiment, type, and news score. Each result is enriched with an AI-generated summary, event classification, publisher metadata, sentiment, company mentions, geography, industry tags and the full article text. Results are returned in the API response.

## Endpoint Details

- **Method:** GET
- **Endpoint:** `/v1/news`

## Authentication requirements

- Include a valid API key in the `x-api-key` request header.

## Request

### Request Parameters

#### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key.
</ParamField>

## URL Parameters

<ParamField query="company" type="string">
  Website (e.g. `https://canva.com`) or company UUID returned by the Company Search API.
</ParamField>

<ParamField query="industry" type="string">
  Industry name to filter articles by.
</ParamField>

<ParamField query="start_date" type="string">
  Start of the date range, formatted as `YYYY-MM-DD`. Non-enterprise plans are limited to a 6-month lookback.
</ParamField>

<ParamField query="end_date" type="string">
  End of the date range, formatted as `YYYY-MM-DD`.

  Default = today
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of news articles to return.

  Default = 10

  Max value = 100
</ParamField>

<ParamField query="offset" type="integer">
  Number of results to skip before returning records. Use with `limit` for pagination.

  Default = 0
</ParamField>

<ParamField query="news_score" type="string">
  Filter by news score tier.

  Default = `all`
</ParamField>

<ParamField query="type" type="string">
  Filter by news type.

  Default = `all`
</ParamField>

<ParamField query="sentiment" type="string">
  Filter by sentiment. Accepted values: `positive`, `negative`, `neutral`, `all`.

  Default = `all`
</ParamField>

## Response

#### Successful Response Fields

Returns a JSON object with the following structure:

<ResponseField name="total" type="integer">
  Total number of articles available based on the filters set
</ResponseField>

<ResponseField name="limit" type="integer">
  Limit applied to the request.
</ResponseField>

<ResponseField name="offset" type="integer">
  Offset applied to the request.
</ResponseField>

<ResponseField name="credits_consumed" type="float">
  Number of credits consumed
</ResponseField>

<ResponseField name="count" type="integer">
  Number of articles returned in the data array.
</ResponseField>

<ResponseField name="data" type="array">
  List of news article objects. See **Article Object Fields** below.
</ResponseField>

#### Article Object Fields

<ResponseField name="title" type="string">
  Headline of the article.
</ResponseField>

<ResponseField name="url" type="string">
  Link to the original article.
</ResponseField>

<ResponseField name="publisher" type="string">
  Name of the publishing outlet.
</ResponseField>

<ResponseField name="published_date" type="string">
  Publication date in ISO 8601 format `YYYY-MM-DD`.
</ResponseField>

<ResponseField name="type" type="string">
  Article type. Example: `Secondary Transactions`, `Equity Fund-Raising`.
</ResponseField>

<ResponseField name="sentiment" type="string">
  Sentiment of the article: `positive`, `negative`, or `neutral`.
</ResponseField>

<ResponseField name="news_score" type="string">
  Relevance and quality score of the article. Values:"high", "medium", "low"
</ResponseField>

<ResponseField name="ai_summary" type="string">
  AI-generated summary of the article.
</ResponseField>

<ResponseField name="full_text" type="string">
  Full body text of the article.
</ResponseField>

<ResponseField name="original_language" type="string">
  Language code of the original article. Example: `en`, `fr`.
</ResponseField>

<ResponseField name="author" type="string">
  Author of the article.
</ResponseField>

<ResponseField name="countries" type="array of strings">
  Country codes mentioned in the article.
</ResponseField>

<ResponseField name="industries" type="array of objects">
  Industries associated with the article. See **Industry Object** below.
</ResponseField>

<ResponseField name="tags" type="array of objects">
  Tags associated with the article. See **Tag Object** below.
</ResponseField>

<ResponseField name="company_mentions" type="array of objects">
  Companies mentioned in the article. See **Company Object** below.
</ResponseField>

#### Industry Object

<ResponseField name="name" type="string">
  Industry name. Example: `Software`.
</ResponseField>

<ResponseField name="is_primary" type="boolean">
  `true` if this is the primary industry of the article.
</ResponseField>

#### Tag Object

<ResponseField name="name" type="string">
  Tag label. Example: `IPO`, `Layoffs`.
</ResponseField>

<ResponseField name="code" type="string">
  Tag code. Example: `ipo`, `layoffs`.
</ResponseField>

<ResponseField name="is_primary" type="boolean">
  `true` if this is the primary tag of the article.
</ResponseField>

#### Company Object

<ResponseField name="uuid" type="string">
  Company UUID.
</ResponseField>

<ResponseField name="name" type="string">
  Company name.
</ResponseField>

<ResponseField name="website" type="string">
  Company website URL.
</ResponseField>

<ResponseField name="is_primary" type="boolean">
  `true` if this is the main company the article is about.
</ResponseField>
