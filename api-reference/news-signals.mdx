## Overview

The News API fetches news articles filtered by company, industry, date range, sentiment, type, and news score. Each result is enriched with an AI-generated summary, event classification, publisher metadata, sentiment, company mentions, geography, industry tags, named entities (people, locations, products, and events mentioned in the article) and the full article text. Results are returned in the API response.

## Endpoint Details

- **Method:** GET
- **Endpoint:** `/v1/news`

## Authentication requirements

- Include a valid API key in the `x-api-key` request header.

## Request

### Request Parameters

#### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key.
</ParamField>

## URL Parameters

<ParamField query="company" type="string">
  Website (e.g. `https://canva.com`) or company UUID returned by the Company Search API.
</ParamField>

<ParamField query="industry" type="string">
  Comma separated list of industry codes to filter articles by. Use [industry search](https://docs.akta.pro/industry-search-api) endpoint to fetch the list of industry codes for a given industry topic.
</ParamField>

<ParamField query="query" type="string">
  Topic name to search articles. Allows open-ended topic name. Eg. Crude oil price developments
</ParamField>

<ParamField query="title" type="string">
  Searches articles based on the title entered.
</ParamField>

<ParamField query="start_date" type="string">
  Start of the date range, formatted as `YYYY-MM-DD`. Non-enterprise plans are limited to a 6-month lookback.
</ParamField>

<ParamField query="end_date" type="string">
  End of the date range, formatted as `YYYY-MM-DD`.

  Default = today
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of news articles to return.

  Default = 10

  Max value = 1000
</ParamField>

<ParamField query="offset" type="integer">
  Number of results to skip before returning records. Use with `limit` for pagination.

  Default = 0
</ParamField>

<ParamField query="unique_article" type="boolean">
  Accepts boolean. True will group similar articles from the same event.
</ParamField>

<ParamField query="news_score_list" type="string">
  Filter by news score tier. Accepted values: `High`, `Medium`, `Low` , `all`

  Default = `all`
</ParamField>

<ParamField query="countries" type="string">
  Filters articles based on the country of the event. Comma separated list of **3 letter ISO country codes**. [View](/country-codes) full list of ISO country codes.
</ParamField>

<ParamField query="blacklisted" type="string">
  Comma separated list of publisher domains to exclude.
</ParamField>

<ParamField query="type_list" type="string">
  Filter by news type. [View](/news-types)  full list of news categories.

  Default = `all`
</ParamField>

<ParamField query="sentiment_list" type="string">
  Filter by sentiment. Accepted values: `positive`, `negative`, `neutral`, `all`.

  Default = `all`
</ParamField>

<ParamField query="entity_person_list" type="string">
  Comma separated list of people names.
</ParamField>

<ParamField query="entity_location_list" type="string">
  Comma separated list of locations.
</ParamField>

<ParamField query="entity_product_list" type="string">
  Comma separated list of product names
</ParamField>

<ParamField query="entity_event_list" type="string">
  Comma separated list of event names
</ParamField>

<ParamField query="naics_code_list" type="string">
  Comma separated list of NAICS codes. [View](/naics-codes) full list of NAICS codes.
</ParamField>

<ParamField query="sic_code_list" type="string">
  Comma separated list of SIC codes. [View](/sic-codes)  full list of SIC codes.
</ParamField>

<ParamField query="iptc_code_list" type="string">
  Comma separated list of IPTC codes. [View](/iptc-codes) full list of IPTC codes.
</ParamField>

<ParamField query="iab_code_list" type="string">
  Comma separated list of IAB codes. [View](/iab-codes) full list of IAB codes.
</ParamField>

## Response

#### Successful Response Fields

Returns a JSON object with the following structure:

<ResponseField name="total" type="integer">
  Total number of articles available based on the filters set
</ResponseField>

<ResponseField name="limit" type="integer">
  Limit applied to the request.
</ResponseField>

<ResponseField name="offset" type="integer">
  Offset applied to the request.
</ResponseField>

<ResponseField name="credits_consumed" type="float">
  Number of credits consumed
</ResponseField>

<ResponseField name="count" type="integer">
  Number of articles returned in the data array.
</ResponseField>

<ResponseField name="data" type="array">
  List of news article objects.
</ResponseField>
